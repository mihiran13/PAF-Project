## 🚀 Quick Start

### Backend
```bash
cd smart-campus
mvn clean install
mvn spring-boot:run
# Backend available at http://localhost:8080
```

### Frontend
```bash
cd smart-campus-frontend
npm install
npm run dev
# Frontend available at http://localhost:5173
```
